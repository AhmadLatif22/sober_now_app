# Sober Now App

A Flutter application for sobriety tracking and support.

## Getting Started

This project is a Flutter application.

### Prerequisites
- Flutter SDK
- Dart SDK

### Installation

1. Clone the repository
```bash
git clone https://github.com/AhmadLatif22/sober_now_app.git
```

2. Navigate to project directory
```bash
cd sober_now_app
```

3. Install dependencies
```bash
flutter pub get
```

4. Run the app
```bash
flutter run
```
